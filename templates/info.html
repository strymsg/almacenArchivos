El botadero

Este es un sitio donde se puede subir archivos para que cualquiera pueda descargarlos, si subes un archivo todos podrán descargarlo durante {{ td0 }} a {{ td2 }} días dependiendo del espacio que ocupa, después de ese tiempo se borrará automáticamente.

¿Por qué se borran los archivos?

La principal razón es para ahorrar espacio de almacenamiento, este servidor ha dispuesto {{ dm }} MB para almacenar archivos, si los archivos no se borrarán automáticamente, el espacio se terminaría tarde o temprano.

Mientras más espacio disponible haya más archivos se pueden compartir y más gente puede descargarlos, además siendo este un espacio de almacenamiento temporal con esto los contenidos que la gente comparte se renueva constantemente.

Criterios de borrado de archivos

Si un archivo subido tiene un tamaño menor que {{ sz1 }} MB se borrará en {{ td0 }} días, si tiene un tamaño entre {{ sz1 }} MB y {{ sz2 }} MB se borrará en {{ td1 }} días, si un archivo tiene un tamaño mayor que {{ sz2 }} MB y menor o igual que {{ ms }} MB se borrará {{ td2 }} días y si el tamaño es mayor que {{ ms }} MB no podrá ser subido.

Cuando un archivo se borra, sus registros en el servidor tambíen se eliminan.

No se guardan archivos duplicados

Si se trata de subir un archivo cuyo nombre existe entre los disponibles para descarga, se cancela la subida.

Si el resultado de hacer sha1sum a un archivo que intentas subir coincide con el sha1sum de otro disponible para su descarga, también se cancela la subida.

No se guardan registros de los usuarios

Cuando subes un archivo, la aplicación no guarda datos relacionados a tu dirección IP o computadora.

Respeta a todas y todos

Recuerda que este es un espacio público y si subes algo todas y todos podrán verlo.

Comparte archivos que sean de utlidad de las y los visitantes de este sitio, no subas material que no les sirva o que les perjudique.

Aunque la aplicación no tiene como detectar esto, si hay contenido que perjudique a alguién los administradores de este servidor podrán borrarlo en cualquier momento.

Cuida este espacio para que siga funcionando por que compartir es bueno.
