El Botadero

"El Botadero" es una "carpeta" pública donde cualquier persona puede subir archivos y descargarlos. Los archivos aquí tienen un límite de permanencia ({{ td0 }} a {{ td2 }} días) pero no límites de velocidad.

¿Por qué se borran los archivos?

La principal razón es para ahorrar espacio de almacenamiento: este servidor ha dispuesto {{ dm }} MB para almacenar archivos (si los archivos no se borrarían automáticamente, el espacio se terminaría tarde o temprano)

Mientras más espacio disponible haya más archivos se pueden compartir y más gente puede descargarlos. Además, siendo este un espacio de almacenamiento temporal, forzando el borrado de contenido viejos estimulamos la renovación del contenido del Botadero

Criterios de borrado de archivos

Usualmente los archivos desaparecen por el límite de tiempo:

  • Archivos menores a {{ sz1 }} MB: {{ td0 }} días
  • Archivos entre {{ sz1 }} MB y {{ sz2 }} MB: {{ td1 }} días
  • Archivos entre {{ sz2 }} MB y {{ ms }} MB: {{ td2 }} días

No se aceptan archivos mayores a {{ ms }} MB.

Cuando un archivo se borra, sus registros en el servidor tambíen se eliminan.

¿Qué pasa con los archivos duplicados?

El Botadero rechaza los archivos subidos que tengan un nombre igual a algún otro que ya exista. También rechaza los archivos que sean exactamente iguales (en contenido) a otro que ya exista, tengan el mismo nombre o no.

No se guardan registros de los usuarios

Cuando subes un archivo, la aplicación no guarda datos relacionados a tu dirección IP o computadora.

Sube los archivos según su categoría

Existen categorías (#) de archivos para mantenerlos clasificados y así es más sencillo que alguien los ubique. Sube archivos en las categorías correspondientes, si no estás seguro súbelo en #Misc.

Ten respeto por los usuarios

Recuerda que este es un espacio público y si subes algo todas y todos podrán verlo.

El Botadero no filtra (aún) por sí solo el contenido: este es moderado continuamente por administradores y estos eliminarán los archivos si estos incumplen alguna de estas reglas.

Contenido prohibido

Aunque el contenido del portal es "vale todo" (documentos, música, etc), existen algunos tipos no admitidos:

  • Archivos que hagan referencia a delitos o actividad ilegal en Bolivia
  • Archivos con contraseña

Cuida este espacio para que siga funcionando por que compartir es bueno.
